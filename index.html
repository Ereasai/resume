MCASM

A custom assembly language created to be used in Minecraft Java Edition. This is a project that I worked on with a friend of mine. The main motivation of this project was to achieve control flow within Minecraft, which was hard to do with vanilla features. Naturally, we created our own assembly language that would allow for branching logic and other useful features, like having variables and complex floating point operations. It was an peculiar use of our knowledge in compilers and programming language.
